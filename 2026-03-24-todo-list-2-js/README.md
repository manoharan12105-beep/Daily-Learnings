# 📝 Todo App V2 (JavaScript)

An improved version of my previous Todo List project, built using **HTML, CSS, and Vanilla JavaScript**.

This version focuses on better **UI/UX, additional features, and cleaner functionality**, making it closer to a real-world application.

---

## 🚀 What’s New in This Version

* ✅ Mark tasks as completed (click to toggle)
* 📊 Task counter (Total & Completed)
* 🎨 Improved UI with card layout and styling
* 🧠 Better data structure using objects instead of plain strings
* 🚫 Duplicate task prevention
* 💾 Persistent storage using **localStorage**

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ✅ Toggle task completion
* 💾 Save tasks in browser (localStorage)
* 📊 View total and completed tasks
* 🎨 Clean and minimal UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Browser localStorage

---

## ⚙️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/manoharan12105-beep/Daily-Learnings.git
```

2. Navigate to the project folder:

```bash
cd 2026-03-24-todo-list-2-js
```

3. Open `ToDo2.html` in your browser.
